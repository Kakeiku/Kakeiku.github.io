QuikJob

QuikJob is a final year Capstone Project I developed alongside two others. This web application was created as a way for employers to find potential staff in a fast and easy way. This is different to your traditional job seeking sites as it suggests you jobs based on what skills you have listed your profile, which is all done with a simple matching algorithm. It also works the other way around by allowing employers to get a list of all job seekers with the skills listed on jobs they have posted.

  • Framework - Ruby on Rails
  • Language - HTML | JavaScript | CSS | Ruby
  • Storage - PostgreSQL
  • Deployment - Heroku
